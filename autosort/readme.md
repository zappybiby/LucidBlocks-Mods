# autosort

adds sort buttons for player and chest inventories plus an autosort settings tab.

install: make a `mods` folder in your game directory if it does not exist, install the qualiamods framework from [qualia mods releases](https://github.com/MarcyMarbles/Qualia-Mods/releases), then drop `autosort.pck` into `mods`. you can tweak how sorting behaves and how the button looks in settings.
