# autosort

adds sort buttons for player and chest inventories.

install: make a `mods` folder in your game directory if it does not exist, install the qualiamods framework from [qualia mods releases](https://github.com/MarcyMarbles/Qualia-Mods/releases), then drop `autosort.pck` into `mods`. configure it from the qualiamods mods menu with `sort_on_open = true|false`, `group_by_type = true|false`, and `small_buttons = true|false`.
