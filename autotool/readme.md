# autotool

automatically selects the best tool for the block you are breaking and can optionally switch to a weapon when attacking entities.

install: make a `mods` folder in your game directory if it does not exist, install the qualiamods framework from [qualia mods releases](https://github.com/MarcyMarbles/Qualia-Mods/releases), then drop `autotool.pck` into `mods`. optional config: set `auto_weapon_on_attack = true|false` in the qualiamods mods menu to let autotool also switch to the strongest melee weapon when you target an entity or npc.
