# hud-details

adds detailed health, tiamana, and biome readouts to the hud.

install: make a `mods` folder in your game directory if it does not exist, install the qualiamods framework from [qualia mods releases](https://github.com/MarcyMarbles/Qualia-Mods/releases), then drop `hud-details.pck` into `mods`. configure it from the qualiamods mods menu with `Health = true|false`, `Tiamana = true|false`, `Biome = true|false`, and `show_level = true|false`.
